### 🚀 Today I Learned

#### | React 환경변수

`환경변수`란, 개발 성격에 따라 분리하는 값

- 환경 변수 설정

```
  REACT_APP_API=/api
```

- 환경 변수 사용

```
  process.env.REACT_APP_API로 접근
```

**실행 환경 분리**

- .env: 기본 설정 파일
- .env.local: .env 파일 override
- .env.development, .env.test, .env.production: 실행 환경에 따라 사용

**예시**

```
  const response = await fetch(`${process.env.REACT_APP_API}/users/logOut`, {
        headers,
        method: "DELETE",
        credentials: "include",
      });
```
